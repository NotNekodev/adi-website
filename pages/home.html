ADI: The Next Generation Driver Interface

Why Choose ADI?

Fully Asynchronous & Event Based

ADI lets the driver immidately process your calls and never blocks threads.

Cross-Platform

ADI lets your drivers to run on all compatible operating systems.

Desktop focused

ADI is void of any server-orianted features like call marshaling over the network.

Fully Public Domain

Don't let Copyright hinder your operating system, ADI is compatible with all software licenses.

What is ADI?

ADI is a powerful driver interface aimed at making drivers work on multiple platforms.

With ADI, developers can write driver for any desktop device and load it on all supported operating systems.

ADI is fully desktop-oriented, allowing you to not worry about server features like call marshaling.
